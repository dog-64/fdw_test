# fdw_test

Тестирование Postgres FDW

Вывод - никакой разницы в разы нет. Всё в пределах погрешности.

## Условия

- Postgres 14.9
- Ubuntu 22.04.2
- Pgbench (14.9 (Ubuntu 14.9-0ubuntu0.22.04.1))

## Создание БД

В create.sql

## Benchmark

Подготовка /единожды

```shell
pgbench -U postgres -i -s 10 fdw_test
```

Запуск

```shell
./benchmark_5m.sh
```
